# Changelog

All notable changes to the Tube Joint Visualizer project.

## [1.0.0] - 2025-11-18

### Added
- Initial release of Tube Joint Visualizer
- Three.js-based 3D visualization engine
- Electron desktop application framework
- Interactive tube creation and manipulation
- Support for rectangular and square tubes
- Customizable tube parameters (width, height, thickness, length)
- Joint angle configuration with presets (30°, 45°, 90°, 135°)
- Multiple tube assembly support
- 3D navigation controls (rotate, pan, zoom)
- Wireframe and solid view modes
- Joint visualization with highlighting
- Undo/redo functionality
- Modern dark theme UI
- Cross-platform build support (Windows, macOS, Linux)
- Comprehensive documentation
