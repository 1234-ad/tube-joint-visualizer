Get the Tube Joint Visualizer running in 3 minutes!

## Prerequisites

- Node.js v16+ ([Download](https://nodejs.org/))
- Git ([Download](https://git-scm.com/))

## Installation (2 minutes)

```bash
# 1. Clone the repository
git clone https://github.com/1234-ad/tube-joint-visualizer.git

# 2. Navigate to directory
cd tube-joint-visualizer

# 3. Install dependencies
npm install
```

## Run the App (30 seconds)

```bash
npm start
```

The application window will open automatically!

## First Steps

1. **Select tube type**: Rectangular or Square
2. **Set dimensions**: Width, height, thickness, length
3. **Click "Add Tube"**: Your first tube appears!
4. **Set angle**: Try 90Â° preset
5. **Add another tube**: It connects at the angle!

## Controls

- **Left-click + drag**: Rotate view
- **Right-click + drag**: Pan camera
- **Scroll wheel**: Zoom in/out

## Build Executable

```bash
# Build for your platform
npm run build

# Find executable in dist/ folder
```
